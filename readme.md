# template-js-tw

A minimal JavaScript + Tailwind CSS template powered by Vite.

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Open in VS Code — accept the recommended extensions prompt
4. Run `npm run dev`

## Adapting for a New Project

1. Update `package.json` — change `name` and reset `version` to `0.1.0`
2. Clear `src/main.js` — remove placeholder code
3. Clear `src/style.css` — keep `@import "tailwindcss"`, remove everything else
4. Replace `public/vite.svg` — swap in your own assets
5. Update `index.html` — change the `<title>` and remove Vite boilerplate
